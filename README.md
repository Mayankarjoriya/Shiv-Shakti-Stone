# Shiv-Shakti-Stone
🪨 StoneCraft – Premium Stone Product Showcase

StoneCraft is a dynamic web application built with Django for displaying and managing premium stone products such as sculptures, garden items, and decorative pieces. The platform provides a smooth user experience with category-based navigation, detailed product information, and an integrated quote request system.

🚀 Features

Category-Wise Display: Products organized under specific categories like Stone Lamp Posts, Garden Furniture, and Planters.

Product Detail Page: Includes material, dimensions, and custom features such as height and size.

Quote Request Form: Users can send inquiries directly from a product’s page.

Email Notifications: Automatic emails are sent to the admin containing customer details and product information in a formatted HTML structure.

Dynamic Media Support: Handles image uploads and displays them seamlessly across the site.

Admin Management: Add, edit, and delete products, categories, and features from the Django admin panel.

Responsive UI: Designed with HTML, CSS, and Bootstrap for all devices.

🧰 Tech Stack

Backend: Django (Python)

Frontend: HTML, CSS, Bootstrap

Database: SQLite / PostgreSQL

Email Service: Django Email Backend

Hosting (Optional): DigitalOcean / Firebase

⚙️ Setup Instructions

Clone the repository:

git clone https://github.com/<your-username>/stonecraft.git
cd stonecraft


Create a virtual environment and install dependencies:

python3 -m venv .env  
source .env/bin/activate  
pip install -r requirements.txt  


Run migrations and start the server:

python manage.py migrate  
python manage.py runserver  


Visit http://127.0.0.1:8000/
 to explore StoneCraft.
